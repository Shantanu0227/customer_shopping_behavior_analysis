# customer_shopping_behavior_analysis
Built an end-to-end customer purchasing behavior analytics project using Python, SQL, and Power BI to uncover business insights through data visualization.


📌 Project Overview

The objective of this project is to analyze customer purchasing behavior and provide meaningful business insights through data visualization.

The project follows a complete ETL and analytics pipeline:


-Data Cleaning & Preprocessing using Python

-Data Storage & SQL Analysis using MySQL

-Interactive Dashboard Development using Power BI

This workflow demonstrates how raw customer transaction data can be transformed into actionable insights for business decision-making.

🧹 Data Cleaning (Python)
